# Seinfeld Conception Calculator

Enter your birthday. Get the Seinfeld episode that was airing when you were (probably) conceived.

Inspired by [this moment](https://youtu.be/41vKRsKISyQ?si=lloEi7D1jUQVmmPe&t=75) from Cinemassacre.

---

**Your birthday never leaves your browser.** It's a single HTML file doing math. You can open the source and check - it's all right there.

The conception window is calculated using the standard full-term range (36–40 weeks from conception). All 180 Seinfeld episodes are hardcoded in the file with their original air dates.
